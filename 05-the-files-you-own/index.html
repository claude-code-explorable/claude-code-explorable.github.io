The files you own in Claude Code

Everything under .claude/ is a markdown file with YAML frontmatter. A skill, a subagent, a rule and a command are almost indistinguishable by looking at them. What separates them is behaviour — when they load, who triggers them, who else ever sees them, and whether Claude is free to ignore them. Most of what you write is advice the model can misread, deprioritise, or lose. A little of it is a gate the model never sees at all. Build a project and find out which is which.
