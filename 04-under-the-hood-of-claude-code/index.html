Under the hood of Claude Code

At its core, Claude Code is a simple while-loop that calls the model, runs tools, and repeats. The model itself is a stateless black box that only emits text. Nearly all the engineering lives in the systems around that loop — context shaping, a deny-first permission gate, delegation. Click through one real turn to see them.

the whole thing, in one glance · every stage below is just detail on one of these lines
while (true) {
  msgs = assemble();          // gather context since the last compaction boundary
  msgs = shape();             // run the five compaction layers to fit the window
  res  = callModel(msgs);     // stream the model's response
  if (noToolUse(res)) break;  // model returned only text → stop
  execute(authorize(res));    // gate, then run the tool calls
}                             // append tool_result, loop again
